Bibliografia

  • GONZALEZ, R.C.; WOODS, R.E.; Processamento Digital de Imagens - 3ª edição. Editora Pearson, 2009.

  • MARQUES FILHO, O.; VIEIRA NETO, H. Processamento digital de imagens. Brasport, 1999.
    • Disponível no site do autor: http://dainf.ct.utfpr.edu.br/~hvieir/pub.html

  • AZEVEDO, E.; CONCI, A.; LETA, F. Computação gráfica: processamento de imagens digitais. Vol.2. Campus-Elsevier, 2008.

  • SOLEM E. Programming Computer Vision with Python. O´REILLY, 2012.
    • Draft (rascunho) disponível para download pelos autores.
    • http://programmingcomputervision.com/

  • VELHO, L.; FRERY, A. C.; GOMES. Image processing for computer graphics and vision, 2 ed., Springer, 2009.

  • SZELISKI, R. Computer Vision: Algorithms and Applications, 2nd ed. Springer, 2020.
    • Draft (rascunho) disponível para download pelos autores.
    • https://szeliski.org/Book/

Ferramentas computacionais

  • Anaconda Distribution
    • Melhor distribuição para trabalhar com projetos usando Python
    • https://www.anaconda.com/distribution/

  • Google Colab
    • Plataforma on-line para execução de códigos do Google. Permite isuar GPU
    • https://colab.research.google.com

  • Scikit-image
    • Biblioteca Python para processamento de imagens
    • https://scikit-image.org/

  • Scikit-learn
    • Biblioteca Python para machine learning
    • https://scikit-learn.org/

  • OpenCv
    • Biblioteca em C++ para Visão Computacional e Processamento de Imagens
    • https://opencv.org/
    • Possui um binding para Python:
      • https://pypi.org/project/opencv-python/
      • https://opencv-python-tutroals.readthedocs.io/en/latest/py_tutorials/py_tutorials.html
