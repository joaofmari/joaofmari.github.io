Instalando e configurando o ambiente de desenvolvimento

Nesta seção serão abordados os principais passos para configuração de um ambiente de desenvolvimento Python em uma máquina local. Nesta seção a enfase será em uma máquina que não dispoe de uma GPU e as informações relavantes para configuração de um ambiente com GPU serão discutidas em outra seção (SECAO).

Nós utilizamos a distribuição Anaconda para Python.
