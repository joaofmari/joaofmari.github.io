PROJETOS:

  • Métodos de segmentação de células em imagens de microscopia baseado na transformada watershed

      • Coordenador: Prof. João Fernando Mari

  • Classificação de Imagens de Células HEP-2 utilizando redes neurais convolucionais aprimoradas por algoritmos evolutivos

      • Coordenador: Prof. Murilo Coelho Naldi

      O objetivo principal deste trabalho é classificar imagens de células HEp-2 utilizando CNNs, explorar CNNs utilizando diferentes estratégias de pré-processamento e treinamento. Adicionalmente, investigar como os algoritmos evolutivos podem aprimorar uma CNN.

  • Aplicação de métodos computacionais para análise de estresse hídrico em plantações por meio imagens térmicas

      • Coordenador: Prof. Tácito Trindade de Araújo Tiburtino Neves

      O principal objetivo deste trabalho é desenvolver um arcabouço de análise de imagens térmicas de áreas de plantação, capaz de identificar a presença ou não de estresse hídrico. Assim, também são objetivos desse projeto: (i) o estudo de algoritmos de processamento de imagens aplicados para segmentação de regiões de plantio em imagens térmicas; (ii) o estudo de algoritmos de extração de características aplicados nas imagens afim de gerar descritores; (iii) o estudo de algoritmos de análise de dados aplicados sobre os descritores para a identificação do nível de estresse hídrico da plantação.

  • Estudo de redes neurais convolucionais para classificação de objetos em imagens de microscopia

      • Coordenador: Prof. João Fernando Mari

      O objetivo deste projeto é desenvolver metodologias para a aplicação de Redes Neurais Convolucionais para solucionar o problema da classificação automática de células em imagens de microscopia. Uma série de aspectos deste tipo de aplicação ainda deve ser analisada. Pretende-se avaliar como se comportam as principais arquiteturas de RNCs na classificação de diferentes conjuntos de imagens de células - com diferentes tamanhos - considerando diversas estratégias de treinamento, como aumento de dados e fine-tuning.

      Financiamento: FAPEMIG
