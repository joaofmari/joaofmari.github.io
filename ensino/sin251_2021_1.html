Ferramentas computacionais

  • Anaconda Distribution
    • Melhor distribuição para trabalhar com projetos usando Python
    • https://www.anaconda.com/distribution/

  • Google Colab
    • Plataforma on-line para execução de códigos do Google. Permite isuar GPU
    • https://colab.research.google.com

  • Scikit-image
    • Biblioteca Python para processamento de imagens
    • https://scikit-image.org/

  • Scikit-learn
    • Biblioteca Python para machine learning
    • https://scikit-learn.org/

  • OpenCv
    • Biblioteca em C++ para Visão Computacional e Processamento de Imagens
    • https://opencv.org/
    • Possui um binding para Python:
      • https://pypi.org/project/opencv-python/
      • https://opencv-python-tutroals.readthedocs.io/en/latest/py_tutorials/py_tutorials.html
